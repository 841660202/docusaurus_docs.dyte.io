# Local User - Events

You can subscribe to data changes on the `useDyteSelector` hook

### Video update

Triggered when the user starts / stops the video using `enableVideo` or `disableVideo`

```js
const videoEnabled = useDyteSelector((m) => m.self.videoEnabled);
```

### Audio update

Triggered when the user starts / stops the audio using `enableAudio` or `disableAudio`

```js
const videoEnabled = useDyteSelector((m) => m.self.audioEnabled);
```
