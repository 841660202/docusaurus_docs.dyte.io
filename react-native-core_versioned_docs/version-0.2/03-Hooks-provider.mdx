# React Hooks

We have two react hooks to consume data and take actions in the meeting

### 1. Meetings Provider

```jsx
const { meeting } = useDyteMeeting();
```

Refer to [documentation for reference](./Reference/DyteClient) on the `meeting` object


```jsx
import { useDyteSelector, useDyteMeeting } from '@dytesdk/react-native-core';

function Meeting() {
  const { meeting } = useDyteMeeting();

  const joinedParticipants = useDyteSelector((m) => m.participants.joined);

  /*
    use joinRoom() method or the setup screen component
    to actually enter the meeting
  */
  useEffect(() => {
    meeting.joinRoom();
  }, [meeting]);

  /// display joined participnats

```

### 2. Data Selector

```jsx
const joinedParticipants = useDyteSelector((meeting) => meeting.participants.joined);
```

You can use the hook to subscribe for updates on any of the [objects](./Reference/DyteClient).
This hook will re-render when the selected data changes.
