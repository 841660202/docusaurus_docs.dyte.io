# Introduction

- 🌐 Built with Web Components.
- 0️⃣ Zero external dependencies.
- 🚀 Blazingly fast. Uses browser provided ShadowDOM.
- 🤏 Small bundle size (<500KB) for the entire UI (less if you use individual components).
- ✨ ShadowDOM provides style encapsulation, your CSS styles wouldn't affect our styles and vice versa.

Install the package from npm:

```bash
npm install @dytesdk/react-ui-kit @dytesdk/react-web-core
```

</TabItem>

<TabItem value="yarn">

Install the package from yarn:

```bash
yarn add @dytesdk/react-ui-kit @dytesdk/react-web-core
```

</TabItem>
</Tabs>

:::note Import Errors

If you get import errors for `@dytesdk/ui-kit` or `@dytesdk/web-core`, install them additonally
:::
