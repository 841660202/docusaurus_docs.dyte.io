# DyteMeetingView

## Properties

| Property name | Data type                    | Use                                                                                       |
| ------------- | ---------------------------- | ----------------------------------------------------------------------------------------- |
| delegate      | id<DyteMeetingViewDelegate\> | Pointer which refers to the object that implements the `DyteMeetingViewDelegate` protocol |

## Methods

| Method name             | Params (with data type)                          | Use                                                                           |
| ----------------------- | ------------------------------------------------ | ----------------------------------------------------------------------------- |
| join                    | config: [DyteMeetingConfig](./connection-config) | Join a Dyte meeting with given config parameters                              |
| toggleAudio             |                                                  | Enable and disable audio stream sharing from the participant to the meeting   |
| toggleVideo             |                                                  | Enable and disable video stream sharing from the participant to the meeting   |
| selfToggleScreensharing |                                                  | Enable and disable screensharing from the participant to the meeting          |
| selfPin                 |                                                  | Pin the current participant video to the grid for everyone in the meeting     |
| selfUnpin               |                                                  | Unpin the current participant video from the grid for everyone in the meeting |
