After integrating Dyte, the next step is to initialize your frontend by installing the SDK and initializing the frontend SDKs with the authentication `token`. Authentication `token` is returned by the [Add Participant API](/guides/livestream/build-livestream-app#step-5-add-participants).

Dyte offers a UI kit for utilizing prebuilt design library components and a Core SDK for building a custom user interface without complex media layers. To view code samples and begin with your preferred SDK, refer to the web and mobile quickstart guides of your choice.
