# DyteChat

Fully featured chat component with image & file upload, emoji picker and
auto-scroll.

```jsx
<Center>
  <DyteChat
    meeting={meeting}
    style={{ height: '480px', maxWidth: '320px', backgroundColor: '#000' }}
  />
</Center>
```
