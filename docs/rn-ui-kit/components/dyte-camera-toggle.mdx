# DyteCameraToggle

A button which toggles your camera.

```jsx
<Center>
  <DyteCameraToggle size="sm" meeting={meeting} />
  <DyteCameraToggle size="lg" meeting={meeting} />
  <DyteCameraToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
