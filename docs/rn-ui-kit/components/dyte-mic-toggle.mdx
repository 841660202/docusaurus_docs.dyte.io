# DyteMicToggle

A button which toggles your microphone.

```jsx
<Center>
  <DyteMicToggle size="sm" meeting={meeting} />
  <DyteMicToggle size="lg" meeting={meeting} />
  <DyteMicToggle variant="horizontal" size="sm" meeting={meeting} />
</Center>
```
