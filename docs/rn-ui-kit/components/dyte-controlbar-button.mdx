# DyteControlbarButton

A skeleton component used for composing buttons.

```jsx
<Center>
  <DyteControlbarButton
    label="Flight Mode"
    icon={airplaneSVG}
    onClick={() => alert('Flight mode toggled')}
    size="sm"
  />
  <DyteControlbarButton
    label="Flight Mode"
    icon={airplaneSVG}
    onClick={() => alert('Flight mode toggled')}
    size="lg"
  />
  <DyteControlbarButton
    label="Flight Mode"
    icon={airplaneSVG}
    onClick={() => alert('Flight mode toggled')}
    variant="horizontal"
    size="sm"
  />
</Center>
```
