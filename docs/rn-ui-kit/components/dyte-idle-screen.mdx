# DyteIdleScreen

A screen that handles the idle state, i.e; when you are waiting for data about
the meeting, specifically the `meeting` object.

```jsx
<DyteIdleScreen
  config={{ designTokens: { logo: '/logo/dark.svg' } }}
  style={{ height: '360px' }}
/>
```
