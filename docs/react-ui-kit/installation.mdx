# Installation

Steps to install packages required for React UI Kit.

Install the package from npm:

```bash npm2yarn
npm install @dytesdk/react-ui-kit @dytesdk/react-web-core
```

:::tip Import Errors

If you get import errors for `zone.js` install it separately

:::
