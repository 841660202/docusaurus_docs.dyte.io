# Installation

## Install with npm

Install the SDK using npm.

```bash npm2yarn
npm install @dytesdk/web-core
```

## Loading via CDN

Add the following script tag in the head of your HTML file.

```html
<script src="https://cdn.dyte.in/core/dyte.js" />
```
