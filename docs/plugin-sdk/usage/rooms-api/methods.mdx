### Get Room ID

Returns the 12 digit unique ID of the room, eg: `afuxxx-rusxxx`.

```tsx
const id = await plugin.room.getID();
```

### Get ID of plugin Initiator

Returns the id of the user that enabled the plugin.

```tsx
const host = await plugin.room.enabledBy();
```

### Get Room Name

Returns the name (meeting title) of the room.

```tsx
const name = await plugin.room.getName();
```

### Get All Peers

Returns a list of all peers in the meeting (excluding self).

```tsx
const peers = await plugin.room.getPeers();
```

### Get Peer

:::info note Returns `undefined` if the peer with given `ID` has left the
meeting or an incorrect `ID` is provided. :::

```tsx
const self = await plugin.room.getPeer();
```

:::tip You can current peer's (self) data by not passing an `ID` to the
function. :::

```tsx
const id = '..,';
const peer = await plugin.room.getPeer(id);
```

### Send Chat Message

Sends a text, image or file message to the meeting.

| Param   | Type        | Description           | Default Value | Required |
| ------- | ----------- | --------------------- | ------------- | -------- |
| message | ChatMessage | Chat message payload. | -             | true     |

```tsx
type ChatMessage =
  | { type: 'text'; message: string }
  | { type: 'image'; image: File }
  | { type: 'file'; file: File };
```

```tsx
const message: ChatMessage = {
  type: 'text',
  message: 'Hello everyone',
};

await plugin.room.sendChatMessage(message);
```
