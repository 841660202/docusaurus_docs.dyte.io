# Background Behaviour

Used to control the behavior of audio/video stream when user puts the app in background.

| Behaviour                 | Meaning                                                                                  |
| --------------------------| ---------------------------------------------------------------------------------------- |
| PICTURE_IN_PICTURE        | App will continue to work in picture in picture mode                                     |
| DISABLE_CAMERA_CAPTURE    | Camera stream will be paused                                                             |
| CONTINUE_CAMERA_CAPTURE   | Won't change anything, app will resume capturing user's video                            |
