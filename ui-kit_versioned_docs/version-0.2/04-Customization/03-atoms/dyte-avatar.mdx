# Avatar

Avatar component is used inside peer view and participant views which displays user picture or user intitials.

You can pass a `participant` object to it to render it properly with a participants data.

```html
<dyte-avatar></dyte-avatar>
```

import { loadSelf } from '/src/utils/meeting';

Sizes `'sm' | 'md' | 'lg'`

<div ref={() => loadSelf('.peerview')} class="flex flex-row space-x-6 items-center">
    <dyte-avatar class="peerview" size="sm" />
    <dyte-avatar class="peerview" size="md" />
    <dyte-avatar class="peerview" size="lg" />
</div>
